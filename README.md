# 💰 Obsidian Finanças

Uma ferramenta simples em Python para registrar gastos e ganhos diretamente no seu Vault do Obsidian, mantendo seus dados localmente

## ✨ Funcionalidades

- Interface gráfica simples (Tkinter).
- Botão de configuração ⚙️ para selecionar o caminho do Vault.
- Adição automática de registros em arquivos `.md`.
- Leve e focado em produtividade.

## 🚀 Como usar (Windows)

1. Vá na aba [Releases](https://github.com/cardosoVg/obsidian-financas/releases).
2. Baixe o arquivo `ObsidianFinancas.exe`.
3. Execute e configure o caminho da sua pasta de finanças no Obsidian.

## 🛠️ Para Desenvolvedores (

```zsh
# Clonar o repositório
git clone https://github.com/cardosoVg/obsidian-financas.git
cd obsidian-financas

# Instalar dependências
pip install -r requirements.txt

# Rodar
python obsidian_financas.py
```
